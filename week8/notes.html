CSS3 Transforms and Transitions

This chapter focuses on two things, the transform property and the translation property. The transform property property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

The translation property The translate() CSS function repositions an element in the horizontal and/or vertical directions.

Other Functions are also mentioned in this chapter as well like the rotate function which rotates an element around the point of origin. It also mentions transitions which allows you to change property values smoothly, over a given duration.
